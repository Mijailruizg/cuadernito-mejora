Manual Contable

1. Principios Contables Básicos

La aplicación sigue los principios de registro cronológico, partida doble y consistencia. Cada movimiento debe estar correctamente clasificado como ingreso o gasto.

2. Registro de Transacciones

  • Registra cada venta, gasto o ingreso usando el formulario principal.
  • Asocia cada movimiento a un rubro y producto específico.
  • El sistema actualiza automáticamente el stock al registrar ventas o gastos.

3. Control de Stock

  • Gestiona el stock de cada producto desde el menú flotante.
  • Puedes importar y exportar el stock y productos usando archivos Excel.

4. Conciliación y Reportes

  • Genera reportes diarios, semanales, mensuales y anuales para revisar ingresos, gastos y saldo.
  • Exporta los movimientos a Excel para conciliación externa o respaldo.

5. Cierre Contable

  • Al final de cada período, utiliza los reportes para realizar el cierre contable y verificar el saldo final.
  • Revisa el movimiento detallado para auditoría y control.
